# 💸 Expense Tracker

A personal finance management app to track daily expenses with ease.

## Features
- ➕ Add & Delete expenses
- 🏷️ Category-based tracking
- 💰 Live total calculation
- 📊 Clean & user-friendly interface

## Tech Stack
- React + Vite
- CSS

## Getting Started
```bash
npm install
npm run dev
```
